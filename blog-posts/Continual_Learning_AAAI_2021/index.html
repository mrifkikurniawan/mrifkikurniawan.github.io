    The annual conference on artificial intelligence conducted by Association for the Advancement of Artificial Intelligence (AAAI) is one of the top-tier conferences publishing, discussing, and exploring the recent advancement of AI on the entire planet. Thereby knowing what has been going on by highlighting what’s matter is undoubtedly essential, at least for myself. Here this article is likely to discuss the selected papers regarding the proposals on continual learning strategy in deep learning.
    Interestingly the published papers in incremental learning range widely such that almost all the common general approaches are related to these newly fresh discoveries. Among those are attempting to build more expressive replay representation , exploring learning dynamics along with novel sampling strategy, exploring the modular disjoint architecture, introducing an uncommon IL-task setting, and even extending IL beyond image classification.

    Continual learning refers to the one of machine learning scenarios in which the data are coming in stream. Consequently, the machine learning model should be able to learn the incoming data in an incremental manner and capable of accommodating the novel concept without forgetting the previous learned knowledge, even getting beneficial from those in order to do the current task. Refer to this article for the continual learning introduction in detail.

    Using Hindsight to Anchor Past Knowledge in Continual Learning

    Approach: Replay/rehearsal
    Scenario: Task incremental learning (task labels are given)
    Paper: link
    Code: link

    What the research is about

    Sampling the small buffer of memory for the incoming replaying stage in continual learning that is most optimum to reduce the catastrophic forgetting is extremely challenging. Aiming to address this issue, the Hindsight Anchor Learning (HAL) not only selected a small buffer of representative samples but also formulated augmenting the rehearsal strategy with additional anchors which designated as learnable anchors or so-called “anchoring” via bi-level optimization problem. Therefore, the key idea here is to finding out most forgettable anchors (one per class per task) through maximizing forgetting of the current trained model of the current task by approximation through fine-tuning the current model to the former episodic memory to maximize forgetting.

    How it works

    The proposed HAL strategy decisive anchors construction of the current training stage in order to minimize forgetting through the entire continual learning experience. The anchors are selected thru gradient ascent in the image space by considering the approximation of the current trained model forgetting anchoring on selected supports on the future task. Nevertheless, given that future learned parameters and the dataset is unavailable in continual learning setting, the forgetting measure is approximated in hindsight which means measuring this forgetting from a fine-tuned predictor on the episodic memory.

    The anchors construction is conducted via measuring forgetting in hindsight or the writer called it as the process of approximating the future by simulating the past. By this it means that the forgetting metric is calculated on task $t$ in the way of fine-tuned the current model $\theta_t$ on past tasks memory $\mathcal{M}$ and resulting transitory model $\theta_{\mathcal{M}}$ followed by measuring the forgetting. After random initialization of anchors in the first stage, the anchors are updated with gradient ascent as follow

    \[e_t \leftarrow e_t + \alpha \nabla_{e_t}\left ( \ell\left ( f_{\theta_{\mathcal{M}}} \left ( e_t, t \right ), y_t \right ) - \ell\left ( f_{\theta_{t}} \left ( e_t, t \right ), y_t \right ) - \gamma\left ( \phi \left ( e_t \right ) - \phi_t \right ) \right )\]

    Which anchors are updated in order to maximize forgetting loss that searching for the anchors maximize the forgetting loss and regularization term of mean embedding loss. Mean embedding loss

    \[\tilde{\theta} \leftarrow \theta - \alpha \nabla_\theta \ell\left ( \mathcal{B} \cup \mathcal{B}_\mathcal{M} \right ),\]

    You can view the complete algorithm description on the paper.

    Interestingly as illustrated in the figure above, the learned anchors seem to belong to the points nearest to the classifier’s decision boundary in the features space. It means that keeping those anchors invariance in the output predictions can reduce the forgetting and maintain the tasks good performance.

    Why it matters

    Episodic memory is well-known as the majority approach underlying the existing state-of-the-art methods in continual learning scenarios. However, as the classes increase while the episodic memory is kept fixed in size, selecting the representative samples into memory becomes crucial. That’s why memory management including performing reasonable useful samples strategy with auxiliary anchors such as this HAL paper proposed is essential as well.
